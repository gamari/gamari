## ショートカット

  - Web上でVSCodeが開く。

## Organization間でのリポジトリの移動

[リポジトリを移譲する](https://docs.github.com/ja/repositories/creating-and-managing-repositories/transferring-a-repository)

1. Settingsを開く。
2. 「Transfer」を押す
3. 移譲先の「Organazation名」を入力する。
